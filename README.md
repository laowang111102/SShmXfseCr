# 行人车辆检测与计数系统 python1076

## 项目概述

行人车辆检测与计数系统是基于Python和YOLOv8深度学习模型开发的应用，具有清新界面，能够实现对图片和视频中等各目标的检测与计数功能。

## 技术栈

- **编程语言：** Python
- **深度学习框架：** YOLOv8
- **其他技术：** 数据可视化、图像处理

## 功能模块

### 检测与计数

- 使用YOL v8算法实现，支持模型的一键切换和更新。
- 能够检测并计数图片和视频中的行人、车辆等目标。

### 实时监控

- 支持摄像头实时检测。
- 方便展示、记录和保存检测结果。

### 目标切换

- 支持在界面中切换不同目标，查看各自的位置。

### 训练与优化

- 提供数据集和训练代码，用户可重新训练和优化模型。

## 系统角色

- **开发者：** 目标检测领域博士在读，专注于目标检测技术的研究与应用。
- **用户：** 需要对行人车辆进行检测和计数的技术人员或研究者。

## 运行环境

- 支持CPU和GPU运行环境。
- 适用于本地安装和远程连接服务器。

## 部署步骤

1. 配置Python环境。
2. 安装YOLOv8深度学习模型。
3. 准备数据集。
4. 运行系统。
